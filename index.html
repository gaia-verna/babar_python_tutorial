Introduction

Python is a very expressive language which means you type less to do more. We have been using Python for our analysis for more than a year. Interactive environment provided by IPython couples with matplotlib allows us to make beautiful plots with quick turnaround. Analysis Tools can take full advantage Python's introspection capability. This enables us to think less about how to get the tools to do what we want and focus more on the analysis. In this tutorial we will be covering how to perform all analysis procedures typically done after skimming process with IPython. Take a look at our advertisement slides.
